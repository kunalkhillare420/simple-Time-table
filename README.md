# Simple Time Table

A simple HTML project that displays the weekly timetable for the IF4K class using an HTML table.

## Features

* Displays the weekly schedule from Monday to Friday.
* Includes class timings and subjects.
* Uses HTML table features such as:

  * `colspan`
  * `rowspan`
  * Table borders and alignment
* Shows a recess period common to all weekdays.
* Displays Saturday as a holiday.

## Technologies Used

* HTML5
